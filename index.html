Como usar:

  1. Clique em "Iniciar Câmera" para ativar a câmera do dispositivo
  2. Posicione o QR code dentro da área destacada
  3. O sistema irá automaticamente detectar e validar o código
  4. Use "Trocar Câmera" para alternar entre frontal e traseira
  5. Clique em "Parar Câmera" quando terminar
